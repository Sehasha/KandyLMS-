# Kandy Library Management System

Kandy Library Management System is a desktop-based application developed to assist librarians and staff in managing daily library operations efficiently.  

The system allows users to add, delete, update, borrow, and return books, as well as calculate borrowing costs. It was built using Python and Tkinter, providing a simple and user-friendly graphical interface to streamline library management tasks.
